# RPG Map Maker

スマートフォン向けのRPGマップ作成ドラフトツール。ブラウザだけで動きます。

**▶ 使う: https://tafuchan.github.io/RPG_Mapmaker/**

## 機能

- **背景1〜3レイヤー**: タイルを塗り重ねられる3枚の背景レイヤー
- **オブジェクトレイヤー**: タイルを自由な位置に配置。ドラッグで移動、2本指ピンチで拡大縮小、複製・反転・前後入れ替え
- **ツール**: ペン / 塗りつぶし(バケツ) / 消しゴム / スポイト、元に戻す・やり直す
- **操作**: 1本指で描画、2本指でパン&ピンチズーム
- **保存**: ブラウザに自動保存。PNG画像書き出し、JSON書き出し/読み込み

ホーム画面に追加するとアプリのように全画面で使えます。

## 構成

- `index.html` / `style.css` / `app.js` — アプリ本体(ビルド不要の静的ページ)
- `assets/` — タイルアトラス(WebP)とメタデータ(`tiles.json`)
- `tilesets/` — 元のタイルシート画像(マゼンタ区切り)
- `tools/slice_tiles.py` — タイルシートからアトラスを再生成するスクリプト

タイルシートを差し替えたら `python tools/slice_tiles.py` で再生成できます(要 Pillow / numpy)。
